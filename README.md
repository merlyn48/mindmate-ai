# MindMate AI – Student Wellbeing Assistant 🌿

MindMate AI is a web-based student wellbeing assistant designed to help students reflect on their mood, track emotional wellbeing, and access supportive resources.

The application provides a simple AI-style chat interface, mood tracking, and mental wellness tools in a calm and user-friendly dashboard.

This project was built using **HTML, CSS, and JavaScript**, with **LocalStorage** used for user session management and data persistence.

---

## Features ✨

### Authentication
- User registration and login system
- LocalStorage-based session management

### Dashboard
- Mood check-in system
- Weekly mood tracker
- Wellbeing activity progress
- Quick access to support tools

### Chat Assistant
- Chat interface for conversations with MindMate
- Multiple conversation sessions
- Messages stored locally per user

### Profile
- User profile with avatar upload
- Profile information stored locally

### Settings
- Theme toggle (light / dark mode)

### Additional Pages
- About
- FAQ
- Contact
- Privacy Policy

### UI Enhancements
- Modern dashboard layout
- Toast notifications
- Sidebar navigation
- Responsive design

---

## Technologies Used 🛠

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **LocalStorage API**
- **Google Fonts (Inter)**

No external frameworks were used to keep the project lightweight and easy to understand.
